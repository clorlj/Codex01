# 🧸 포켓몬 인형 쇼핑몰 (PokePlush)

파란색(Blue) 테마를 바탕으로 제작된 포켓몬스터 캐릭터 인형 전용 온라인 쇼핑몰 웹사이트입니다. HTML, CSS, JavaScript만을 활용하여 구현되었으며, JSON 데이터를 통한 동적 상품 렌더링 및 외부 API 연동을 포함하고 있습니다.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Data:** JSON
- **API:** Kakao Map API (지도), Weather API (날씨 정보)

## 🎨 Design Theme
- **Main Color:** Blue (파란색)
- **Concept:** 시원하고 신뢰감을 주는 파란색을 메인 테마로 하여, 귀여운 포켓몬 인형들이 돋보일 수 있는 깔끔한 UI를 지향합니다.

## ✨ Key Features

1. **동적 화면 전환 (SPA 방식 네비게이션)**
   - 4개의 서브메뉴(예: `홈`, `상품 보기`, `투데이`, `오시는 길`)로 구성된 네비게이션 바를 제공합니다.
   - 각 서브메뉴 클릭 시 페이지 새로고침 없이 JavaScript를 통해 해당 콘텐츠만 화면에 동적으로 표시됩니다.

2. **JSON 기반 상품 갤러리**
   - `상품 보기` 탭 클릭 시, 로컬 `json` 파일에 구축된 포켓몬 인형 데이터를 JavaScript `fetch`로 불러옵니다.
   - 불러온 데이터를 바탕으로 상품 이미지, 이름, 가격 등의 목록을 화면에 렌더링합니다.

3. **실시간 정보 위젯 (날짜, 시간, 날씨)**
   - 웹사이트 상단 또는 특정 메뉴(`투데이` 탭 등)에서 사용자가 접속한 시점의 **현재 일자와 시간**이 실시간으로 업데이트되어 표시됩니다.
   - 현재 **오늘의 날씨** 정보를 가져와 화면에 함께 제공합니다.

4. **오프라인 매장 안내 (Kakao Map 연동)**
   - `오시는 길` 메뉴를 통해 쇼핑몰의 오프라인 매장 위치를 안내합니다.
   - **카카오맵 API**를 연동하여 확대/축소 및 길찾기가 가능한 인터랙티브 지도를 제공합니다.
